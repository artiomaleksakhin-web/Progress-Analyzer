# Анализатор успеваемости

Практическая работа по Python. Программа загружает данные студентов из CSV-файла, рассчитывает средний балл, определяет статус успеваемости и выводит статистику по группе.

## Состав проекта

- `analyzer.py` — основной файл программы;
- `students.csv` — входные данные в формате CSV;
- `students.xlsx` — входные данные в формате Excel;
- `requirements.txt` — зависимость для чтения Excel-файла;
- `sample_output.txt` — пример результата выполнения программы;
- `Отчет_Анализатор_успеваемости.docx` — отчёт по практической работе.

## Запуск

1. Установить зависимость для работы с Excel:

```bash
pip install -r requirements.txt
```

2. Запустить программу:

```bash
python analyzer.py
```

По умолчанию программа читает файл `students.csv`. Для проверки чтения Excel-файла в функции `main()` можно заменить строку:

```python
students = load_students_from_csv("students.csv")
```

на строку:

```python
students = load_students_from_excel("students.xlsx")
```
